#Patika Store

Patika.dev sitesinin React Native dersleri kapsamında verdiği ödevlerden biri olan Patika Store uygulamasının çalışır halidir. Aşağıda gördüğünüz ekran görüntüleri Android ve iOS sanal cihazlarına ait ekran görüntüleridir.

Ödev de arama işlevinin çalışmasına gerek yok dense de, çalışır hale getirilmiştir.

Ödevin linki : https://academy.patika.dev/tr/courses/react-native/odev_1
